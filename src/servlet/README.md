# APP COMIDAS

## Descripción

Aplicación web desarrollada en Java que permite validar el acceso de usuarios mediante inicio de sesión.

## Tecnologías utilizadas

* Java
* JSP
* Servlets
* MySQL
* Apache Tomcat

## Módulo desarrollado

* Inicio de sesión de usuarios.

## Funcionalidades

* Conexión a base de datos.
* Validación de usuario y contraseña.
* Acceso al sistema.
* Página de bienvenida.
